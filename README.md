# omkhapekar.github.io-portfolio
# Personal Portfolio Website

A modern, responsive portfolio website built using HTML, CSS, and JavaScript with interactive animations, 3D elements, and a clean UI design.
🚀 Live Demo: https://omkhapekar09-bit.github.io/omkhapekar.github.i-portfolio/

📌 Features

 ✅ Fully Responsive (Mobile + Desktop)
 🎨 Modern UI with Neon Theme
 🍔 Hamburger Menu Navigation
 🌌 Animated Star Background (Canvas API)
 🧊 3D Rotating Cube (Three.js)
 📜 Smooth Scroll Animations
 💼 Project Showcase Section
 📩 Contact Form UI

🛠️ Technologies Used

HTML – Structure of the website
CSS – Styling and responsiveness
JavaScript – Interactivity and logic
Canvas API – Background animation
Three.js – 3D graphics (rotating cube)
Google Fonts – Typography

 📂 Project Structure
 
portfolio-website/
index.html      # Main HTML file
style.css       # Styling file
script.js       # JavaScript functionality
README.md       # Project documentation

⚙️ How It Works

* The website is divided into sections like **Home, About, Skills, Projects, and Contact**.
* A hamburger menu is used for navigation across all screen sizes.
* Canvas API is used to create an animated star background.
* Three.js is used to render a rotating 3D cube.
* JavaScript handles:

  * Menu toggle
  * Scroll animations
  * Responsive resizing
    📱 Responsiveness

This website is fully responsive using:

* Flexible layouts 
* Dynamic font sizing 
* Media-friendly spacing

It works smoothly on:

* 📱 Mobile devices
* 💻 Laptops
* 🖥️ Desktop screens
